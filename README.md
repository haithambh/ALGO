## Project Description
هذا نظام إدارة مكتبة رقمية بسيط تم تطويره بلغة جافا باستخدام مبادئ البرمجة الشيئية (التغليف، والوراثة، وتعدد الأشكال، والواجهات).

## How to Run
1. ترجمة الملفات عن طريق:
    javac src/*java -d comp 
    
2. Run the application:
    java -cp comp src.LibraryApp
##  Features
- Add Book
- Add Borrower       
- Borrow Book        
- Return Book        
- Search Book        
- Show Borrowed Books



## Project Structure
- `src/` : source code
- `docs/` : documentation
